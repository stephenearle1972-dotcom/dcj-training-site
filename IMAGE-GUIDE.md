## üí° Tips

1. **Use lowercase letters** - Better for web compatibility
2. **Use hyphens (-)** not spaces - Spaces cause problems in URLs
3. **Keep .jpg extension** - Works on all browsers
4. **Optimize before upload** - Use tinypng.com to reduce file size
